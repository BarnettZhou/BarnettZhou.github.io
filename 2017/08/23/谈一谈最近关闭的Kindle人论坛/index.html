谈一谈最近关闭的Kindle人论坛

发表于 2017-08-23

原文发表于2015年2月26日，原文地址在本人的博客园博客。

最近Kindle圈子内最大的论坛“Kindle人”关闭了，倒也掀起了一阵小波澜。Kindle人论坛是K友圈子里比较著名的一个“Kindle资源分享论坛”，这么一说其实混了这么久网络，大家都知道这个论坛是怎么回事，我也就不多提了。

本来嘛，经过这么多年的习惯养成，大家在网络上寻找盗版已经成了一种司空见惯的事情。今天我不想讨论继“九月不撸”行动以及关闭人人影视之后，关闭“Kindle人论坛”其中所包含的政治意味，单讨论关闭这个论坛是否合理，以及说一些你们可以跳过的废话。另，这篇文章有的观点我在其他网络社区以匿名方式表达过。

三个问题

先回答以下几个问题以及反驳几个在网络社区上出现的观点：

亚马逊上购买的电子书是否可以像纸质书一样有传播的权利？

我看到很多人在Kindleren被关闭之后表达了这样的观点：这本书是我买来的，为什么不可以借给朋友看？

然而，事实上，你向亚马逊支付的电子书费用并不是用于购买这本书，而是购买了“在你的设备上阅读这本书的凭证”。我们看看亚马逊官方是怎么说的：亚马逊 帮助: Kindle商店使用条件

  • Kindle内容的使用

在您下载Kindle内容并支付任何相关费用（包括相关税费）之后，内容提供商即授予您非排他性权利，您可以无限次地阅读、使用及显示此等Kindle内 容，但该等使用仅限于在Kindle、Kindle阅读软件上或作为服务一部分所允许的方式进行，并且仅限于个人使用，不得用于任何商业用途。 Kindle内容由内容提供商向您提供许可，而非出售给您。内容提供商可能在其Kindle内容中包含其他使用条件。这些条件也将适用，但是，如果出现冲突，则以本协议为准。有些内容可能无法通过Kindle阅读软件向您提供。

  • 限制

除非另外明确说明，您不得向任何第三方出售、出租、租赁、分发、传播、分许可或以其他任何方式转让对Kindle内容或其任何部分享有的任何权利，而且您不得移除或修改Kindle内容上任何专有权告示或标识。此外，您也不得避开、修改、破坏或者绕过保护Kindle内容的安全功能。

  • 损失风险

Kindle内容的损失风险在您下载或获取Kindle内容时转移给您。
重要的内容我已经使用粗体标出，官方说得很清楚，你所购买的电子书只是一个提供商给你的凭证，表示你可以通过亚马逊下载该书籍并在自己的设备上阅读，并不是你购买了这本电子书。假设你想要“传播”这本电子书，可以将你的设备暂借给你的亲友，但是此时电子书的凭证的和Kindle设备依旧属于你个人。

其实我们可以回过头来再看看纸质书，假设你购买了一本纸质书，并且将其借给朋友阅读，这个行为其实与“购买”了Kindle电子书并且将设备结构朋友的行为一致，但如果复制正版图书并且传播，一样算作盗版。盗版_百度百科

中国亚马逊上的电子书质量太差能否构成下载盗版的理由？

盗版就是盗版，就算你说成花还是盗版。正版的存在并不是亚马逊和作者跑出来敛财，而是对于作者写作以及亚马逊排版的回报。用户当然可以觉得亚马逊的排版太烂，甚至可以发邮件给亚马逊官方骂他们（当然骂人是不可取的），但是跑去下载盗版的行为就是在偷盗。

另外，亚马逊提供了部分书籍试读的功能以及7天内退货退款的服务（Kindle电子书退货），可以满足用户想要“鉴别”烂书的需求。这样还要使用盗版就实在说不过去了。

亚马逊上的电子书贵吗？

有的人以自己财力不足作为使用盗版的挡箭牌，那我们来看看亚马逊上的电子书到底贵不贵。

亚马逊每天会推出两本特价电子书，现在春节特惠，每天推出四本，均价6元左右。假设你每周只能看上其中的一本，一年就是52本了，一年共花费312元，这个阅读量已经相当不错了，价格呢也就是和妹子看电影、晚饭（味千拉面之类的便饭）加开房的水平。少开一次房，多读一年书。

再说个题外话，软件。

Office 2013 专业增强版 + Windows 8 专业版 套装这么一套，大一学生购买是849，当然你也可以选择只购买系统，办公软件选用WPS等免费软件。Microsoft Windows 8 专业版光买一个系统是448元，按照我们学校勤工俭学的标准，两个月可入，如果生活费富足，省一省很快就有了。至于IDE，Visual Stuio 2013学生可以免费使用，使用学校邮箱注册或者提供相关证明即可，巨硬简直良心。

可能我这样的说法会被指责为“何不食肉糜”，但是我说的是针对大多数人而言，电子书和正版软件的消费并不昂贵。

所以说，不管是有工资的工薪阶层还是没工资的学生，使用盗版都说不过去。

之前自己是盗版用户就不应该指责现在的盗版用户吗？

很多人持有这样的观点：“你能保证你用的系统是正版，从小到大看过的书都是正版？做到了再来指责我们吧！”看到这样的流氓观点其实挺无奈的，因为根本不值得一驳。这个观点的核心就是想要将维护正版的用户拉到和盗版用户一个层次上来，然后对其进行道德上的攻击，譬如指责正版用户“装逼”、“充满了优越感”之类。道理上说不过你，就企图从道德上击败你，这也是国人长期的思辩路线。

我可以光明磊落地承认，我自己曾经也是盗版用户，我在Kindle人上也下载过盗版的电子书，但是这并不妨碍我之后开始在亚马逊上购买正版的电子书“赎罪”。今后我也会用自己的行动维护正版、尽量做到生活工作全正版（毕竟employer用盗版吃官司的是公司和我没半毛钱关系……）。

我作为一个曾经的盗版用户自然是心中有愧的，但是一直使用盗版的人却振振有词我不配指责他们，这实在是太可笑了。

结语

好了，大致就是这样，下面就是纯属个人发泄时间了。

我对这事怎么看呢，关闭论坛对于保护正版有利，但是这件事也没什么代表意义，也不过是众多Kindle盗版拥簇被蚊子盯了一口的水平。找个片都能直接百度“xxx.torrent”，找本无害的电子书还不容易，不要小看国人的智慧。这么说吧，我专业是计算机科学与技术，我们专业起码一半的人毕业之后会去当程序员，也就是说，今后他们要靠自己码出来的代码吃饭。但是现实是，我们专业也有不少的人持有“买个iPhone那么贵为什么App还要收钱”的观点，怎么说呢，我觉得大家要是都这么想我们还是一起饿死吧。

况且，关闭众多的盗版网站，到底是为了保护版权还是有其他的原因，这就不得而知了。

反正我是不乐观。
